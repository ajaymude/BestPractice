


1. Layout Utilities
These utilities control the layout and structure of elements.

Container
container: Centers the content and sets a max-width based on breakpoints.

Display
block, inline-block, inline, flex, inline-flex, grid, inline-grid, hidden: Controls the display behavior of an element.

Positioning
static, fixed, absolute, relative, sticky: Controls the positioning of an element.

top-0, right-0, bottom-0, left-0: Positions an element relative to its container.

inset-0: Positions an element to cover its entire container.

Z-Index
z-0 to z-50: Controls the stacking order of elements.

Floats
float-right, float-left, float-none, clearfix: Controls floating behavior.






2. Flexbox Utilities
These utilities are used for creating flexible layouts.

Flex Direction
flex-row, flex-row-reverse, flex-col, flex-col-reverse: Controls the direction of flex items.

Flex Wrapping
flex-wrap, flex-wrap-reverse, flex-nowrap: Controls whether flex items wrap.

Alignment
justify-start, justify-end, justify-center, justify-between, justify-around, justify-evenly: Aligns flex items along the main axis.

items-start, items-end, items-center, items-baseline, items-stretch: Aligns flex items along the cross axis.

content-start, content-end, content-center, content-between, content-around, content-stretch: Aligns flex lines within a flex container.

Flex Items
flex-1, flex-auto, flex-initial, flex-none: Controls how flex items grow or shrink.

order-1 to order-12: Controls the order of flex items.





3. Grid Utilities
These utilities are used for creating grid-based layouts.

Grid Template
grid-cols-1 to grid-cols-12: Defines the number of columns in a grid.

grid-rows-1 to grid-rows-6: Defines the number of rows in a grid.

Gap
gap-0 to gap-96: Defines the gap between grid items.

gap-x-*, gap-y-*: Defines horizontal and vertical gaps.

Placement
col-start-*, col-end-*: Places grid items in specific columns.

row-start-*, row-end-*: Places grid items in specific rows.

col-span-*, row-span-*: Spans grid items across columns or rows.






4. Spacing Utilities
These utilities control padding, margin, and spacing.

Padding
p-0 to p-96: Adds padding to all sides.

px-*, py-*: Adds horizontal or vertical padding.

pt-*, pr-*, pb-*, pl-*: Adds padding to specific sides.

Margin
m-0 to m-96: Adds margin to all sides.

mx-*, my-*: Adds horizontal or vertical margin.

mt-*, mr-*, mb-*, ml-*: Adds margin to specific sides.






5. Sizing Utilities
These utilities control the width and height of elements.

Width
w-0 to w-full: Sets the width of an element.

w-screen: Sets the width to the viewport width.

min-w-*, max-w-*: Sets minimum and maximum widths.

Height
h-0 to h-full: Sets the height of an element.

h-screen: Sets the height to the viewport height.

min-h-*, max-h-*: Sets minimum and maximum heights.






6. Typography Utilities
These utilities control the appearance of text.

Font Family
font-sans, font-serif, font-mono: Sets the font family.

Font Size
text-xs to text-9xl: Sets the font size.

Font Weight
font-thin, font-extralight, font-light, font-normal, font-medium, font-semibold, font-bold, font-extrabold, font-black: Sets the font weight.

Text Alignment
text-left, text-center, text-right, text-justify: Aligns text horizontally.

Text Color
text-{color}-{shade}: Sets the text color (e.g., text-red-500).

Text Decoration
underline, line-through, no-underline: Controls text decoration.

Text Transform
uppercase, lowercase, capitalize, normal-case: Transforms text.

Line Height
leading-3 to leading-10: Sets the line height.






7. Background Utilities
These utilities control the background of elements.

Background Color
bg-{color}-{shade}: Sets the background color (e.g., bg-blue-500).

Background Image
bg-none, bg-gradient-to-{direction}: Sets the background image or gradient.

Background Position
bg-{position}: Sets the background position (e.g., bg-center).

Background Size
bg-cover, bg-contain: Sets the background size.






8. Border Utilities
These utilities control the borders of elements.

Border Width
border, border-0, border-2, border-4, border-8: Sets the border width.

Border Color
border-{color}-{shade}: Sets the border color (e.g., border-gray-300).

Border Radius
rounded, rounded-sm, rounded-md, rounded-lg, rounded-full: Sets the border radius.






9. Effects Utilities
These utilities control shadows and opacity.

Box Shadow
shadow-sm, shadow, shadow-md, shadow-lg, shadow-xl, shadow-2xl, shadow-inner, shadow-outline, shadow-none: Adds box shadow.

Opacity
opacity-0 to opacity-100: Sets the opacity of an element.






10. Interactivity Utilities
These utilities control user interaction.

Cursor
cursor-auto, cursor-pointer, cursor-wait, cursor-not-allowed: Sets the mouse cursor.

Pointer Events
pointer-events-none, pointer-events-auto: Controls pointer events.

User Select
select-none, select-text, select-all, select-auto: Controls text selection.






11. Transition and Animation Utilities
These utilities control transitions and animations.

Transition
transition-none, transition-all, transition, transition-colors, transition-opacity, transition-shadow, transition-transform: Controls transitions.

Duration
duration-75 to duration-1000: Sets the transition duration.

Easing
ease-linear, ease-in, ease-out, ease-in-out: Sets the transition timing function.

Delay
delay-75 to delay-1000: Sets the transition delay.






12. Transform Utilities
These utilities control transformations.

Transform
transform, transform-gpu, transform-none: Enables transformations.

Translate
translate-x-*, translate-y-*: Translates an element.

Scale
scale-*, scale-x-*, scale-y-*: Scales an element.

Rotate
rotate-*: Rotates an element.

Skew
skew-x-*, skew-y-*: Skews an element.






13. Customization Utilities
These utilities allow for customization.

Arbitrary Values
w-[value], h-[value], text-[value], etc.: Allows custom values (e.g., w-[500px]).

Custom Colors
bg-[#hex], text-[#hex], etc.: Allows custom colors (e.g., bg-[#ff0000]).
