concise list of MongoDB data types in a single screen:

Null: Represents a null value.
Boolean: Represents a true/false value.
Integer: 32-bit signed integer.
Long: 64-bit signed integer.
Double: Double-precision floating point number.
Decimal128: 128-bit decimal floating point for high precision.
String: UTF-8 encoded text.
ObjectId: 12-byte unique identifier.
Binary Data: Raw binary data (e.g., BinData).
Date: UTC date-time.
Timestamp: 64-bit timestamp used for internal purposes.
Regular Expression: Pattern for text search.
Array: Ordered list of values.
Embedded Document: Nested document or subdocument.
Object: A document (can contain nested objects or arrays).
MinKey: A special value for comparison to the lowest value.
MaxKey: A special value for comparison to the highest value.
JavaScript: JavaScript code.
JavaScript with Scope: JavaScript code with variable scope.
