1. Methods for Getting Date and Time Values
These methods retrieve various parts of a date and time.

Date and Time Values:
getDate() ‚Äì Returns the day of the month (1-31).
getDay() ‚Äì Returns the day of the week (0-6, where 0 is Sunday).
getFullYear() ‚Äì Returns the full year (e.g., 2025).
getHours() ‚Äì Returns the hour (0-23).
getMilliseconds() ‚Äì Returns the milliseconds (0-999).
getMinutes() ‚Äì Returns the minutes (0-59).
getMonth() ‚Äì Returns the month (0-11, where 0 is January).
getSeconds() ‚Äì Returns the seconds (0-59).
getTime() ‚Äì Returns the number of milliseconds since January 1, 1970, 00:00:00 UTC.
getTimezoneOffset() ‚Äì Returns the time difference between UTC and the local time, in minutes.
UTC Date and Time Values:
getUTCDate() ‚Äì Returns the day of the month in UTC (1-31).
getUTCFullYear() ‚Äì Returns the full year in UTC.
getUTCMonth() ‚Äì Returns the month in UTC (0-11).
getUTCDay() ‚Äì Returns the day of the week in UTC (0-6).
getUTCHours() ‚Äì Returns the hour in UTC (0-23).
getUTCMinutes() ‚Äì Returns the minutes in UTC (0-59).
getUTCSeconds() ‚Äì Returns the seconds in UTC (0-59).
getUTCMilliseconds() ‚Äì Returns the milliseconds in UTC (0-999).
2. Methods for Setting Date and Time Values
These methods allow you to set specific parts of a date and time.

Date and Time Values:

setDate(day) ‚Äì Sets the day of the month (1-31).
setFullYear(year, month, day) ‚Äì Sets the full year, month, and day.
setHours(hour, minute, second, millisecond) ‚Äì Sets the hour, minute, second, and millisecond.
setMilliseconds(milliseconds) ‚Äì Sets the milliseconds (0-999).
setMinutes(minute, second, millisecond) ‚Äì Sets the minute, second, and millisecond.
setMonth(month) ‚Äì Sets the month (0-11).
setSeconds(second, millisecond) ‚Äì Sets the second and millisecond.
setTime(milliseconds) ‚Äì Sets the time in milliseconds since January 1, 1970.
UTC Date and Time Values:

setUTCDate(day) ‚Äì Sets the day of the month in UTC (1-31).
setUTCFullYear(year, month, day) ‚Äì Sets the full year, month, and day in UTC.
setUTCMonth(month) ‚Äì Sets the month in UTC (0-11).
setUTCHours(hour, minute, second, millisecond) ‚Äì Sets the hour, minute, second, and millisecond in UTC.
setUTCMinutes(minute, second, millisecond) ‚Äì Sets the minute, second, and millisecond in UTC.
setUTCSeconds(second, millisecond) ‚Äì Sets the second and millisecond in UTC.
setUTCMilliseconds(milliseconds) ‚Äì Sets the milliseconds in UTC.
3. Methods for Formatting Dates and Times
These methods return the date and time in various string formats.

Standard Formatting:
toDateString() ‚Äì Returns the date portion as a readable string.
toTimeString() ‚Äì Returns the time portion as a readable string.
toISOString() ‚Äì Returns the date and time in ISO 8601 format (e.g., "2025-01-31T14:30:15.000Z").
toJSON() ‚Äì Returns a JSON string representation of the date (same as toISOString()).
toLocaleDateString() ‚Äì Returns the date in the local date format.
toLocaleTimeString() ‚Äì Returns the time in the local time format.
toLocaleString() ‚Äì Returns both date and time in the local format.
toString() ‚Äì Returns a string representing the date and time.
toUTCString() ‚Äì Returns the date and time in UTC format.
4. Methods for Date Arithmetic
These methods help with date calculations and operations.

Date.now() ‚Äì Returns the current time in milliseconds since January 1, 1970.
Date.parse(dateString) ‚Äì Parses a date string and returns the corresponding time in milliseconds.
Date.UTC(year, month, day, hours, minutes, seconds, milliseconds) ‚Äì Returns the number of milliseconds for a given UTC date.
Date.prototype.valueOf() ‚Äì Returns the primitive value of a Date object (i.e., the number of milliseconds since January 1, 1970).
5. Static Methods
These methods are available on the Date constructor itself, rather than instances of Date.

Date Parsing and Conversion:
Date.parse(dateString) ‚Äì Parses a date string and returns the time in milliseconds.
Date.UTC(year, month, day, hours, minutes, seconds, milliseconds) ‚Äì Returns the time in milliseconds for a UTC date.
